== Description ==
Love the ease with which you can make changes to your WordPress site with the built-in Customizer tool? Don't like the standard look of the WooCommerce emails too much? This WooCommerce email customizer extension is exactly what you need!

This plugin does only one thing but does it well - it sets up a separate instance of the Customizer tool and allows you to make changes to WooCommerce emails with live preview.

 &#9989;  Tested OK with WooCommerce 4.5.2.

Customize your WooCommerce emails now and stand out from the crowd!

Created at [Webtoffee](https://www.webtoffee.com/plugins) by people who know what they are doing.
