== Description ==
Woo Custom Related Products for WooCommerce allows you to choose related products for the particular product.

= Plugin Functionality: =
* Add custom related products.
* Display chosen related products on product detail page instead of display same categories products.

== Installation ==
* Download the plugin
* Upload the folder "woo-custom-related-products" to wp-content/plugins (or upload a zip through the WordPress admin)

== Frequently Asked Questions ==
= How to Setup : =
* Step-1: Go to edit product section in product data section go to Linked Products you will find 'Custom Related Product'.
* Step-2: Select products which you want to set related products for that product.
* Step-3: Save product.
